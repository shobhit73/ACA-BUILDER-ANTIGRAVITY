# Form 1095-C PDF Template

Place the blank IRS Form 1095-C PDF file here with the filename:
`form-1095-c-2024.pdf`

You can download the official blank form from:
https://www.irs.gov/pub/irs-pdf/f1095c.pdf

The PDF generation system expects this file to be present at:
`/forms/form-1095-c-2024.pdf`

Make sure the PDF has fillable form fields (AcroForm fields) for the system to populate.
